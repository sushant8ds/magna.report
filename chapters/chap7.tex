\chapter{TESTING}

\justify

This chapter describes the testing strategy for the HPDC Defect Detection System at two levels: acceptance testing (end-to-end system performance against defined success criteria) and unit testing (validating individual pipeline modules in isolation).

\section{Test Plan and Test Cases}

\textbf{Types of testing performed:}
\begin{itemize}[leftmargin=2em]
    \item \textbf{Acceptance Testing:} End-to-end system validated against the minimum mAP50-95 threshold and functional requirements from the SRS.
    \item \textbf{Unit Testing:} Each pipeline module tested independently with known inputs and verified outputs.
\end{itemize}
